À PROPOS DU PROJET



Notre projet tuteuré est basé sur la transmission en temps réel de flux de données en streaming (vidéo et audio).

Ce projet est né de l'intérêt que porte l’un des membres du groupe pour les technologies de streaming.

Notre objectif théorique est aller de pouvoir créer notre petite architecture sur un réseau comme Twitch peut le faire sur Internet.
